Paddle Champion

Paddle Champion

API Rest hecha con Spring Boot 2.7.4, base de datos MySQL con JPA Hibernate. Diseñada para armar, gestionar y organizar torneos de paddle. Asegurada con Spring Security y Json Web Token tiene un sistema de autorizacion mediante roles. Puedes crear torneos y agregar equipos a ellos, contiene un algoritmo que crea todos los partidos automaticamente y arma un fixture en base a estos.
